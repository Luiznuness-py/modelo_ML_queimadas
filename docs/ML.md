# Machine Learning - Aprendizado Supervisionado

Assim como foi alinhado anteriormente será utilizado algoritmos de ML que utilizam o conceito de aprendizado supervisionado.

No aprendizado supervisionado, treinamos o modelo com um conjunto de dados de entrada e um conjunto correspondente de dados de saída.

## Tipos de Algoritmos

Para esse tipo de aprendizado temo diferentes tipos de algoritmos, e para alguns temos a possibilidade de ter dois tipos de retorno, sendo eles:

- **Classificação**: você passa um rótulo pré definido de alguns valores semelhantes de saída e assim o modelo retornará um desses valores.
- **Regressão Linear**: você vai passar os dados de saída assim como o de entrada e o modelo retornará o valor concreto.

## Objetivo do Modelo

Para você criar um modelo de ML usando um algoritmo de aprendizado supervisionado você tem que passar um relação de parâmetros de entrada e um único parâmetro de saída, a algoritmo vai buscar entender os padrões de entrada para descobrir o parâmetro de saída.

Então o nosso objetivo é um só, passar os parâmetros certos para que o modelo consiga entender os padrões e retorno os valores mais corretos possíveis.
